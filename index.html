Welcome to your first Angular app!

What's the single worse feeling in the world? Answer: When disaster strikes, and you need to call a superhero, but you've forgot their phone number. THE WORST FEELING IN THE WORLD.

That's why you're going to build: the superhero phone book!

In this repository, you're going to find just three files: an index.html, app.js in the js folder, and app.css in the css folder.

Your job is to build an app that helps you find superheroes's phone numbers. You app will need to display the list of superheroes in app.js, allow you to search that list, and color code the list by whether the superhero is marvel, dc, or independent. You'll also want to add a button next to each superhero to display an image for the superhero.

To accomplish your goal, you'll need to:

  1. Bootstrap your angular application. ngApp Documentation, Angular Tutorial - Bootstrapping
  2. Setup a simple module and a controller, attach a controller to your view. module Documentation, Controller documentation, Angular Tutorial Step 2
  3. Use your controller to add the list of superheroes to the scope. Use ng-repeat to render the list of superheroes. NgRepeat documentation, also see step 2 of tutorial above
  4. Use ng-model and ng-filter to add a search box to your superhero list. Step 3 of Angular Tutorial
  5. Use ng-class to add a class to each super hero in the list to color code the list by which comic series the superhero belongs to. The classes are predefined in app.css but feel free to add to them.
  6. Add a button next to superhero to display their image at the top of the page. (image URLs are in the list) Angular Tutorial Step 10, NgClick Documentation

Bonus Challenges (if you finish):

  1. Add a small form to add superheroes to your list.Form documentation
  2. Use ngRoute to switch between a detail and a list view. Angular Tutorial Step 7, ngRoute documentation
  3. Use AngularFire to load your superheroes from a remote database. Pre-entered list is hosted at superherophonebook.firebaseapp.com. AngularFire Documentation
